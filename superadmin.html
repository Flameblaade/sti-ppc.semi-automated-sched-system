Data Management

Backup Data

Export all system data including users, departments, programs, subjects, rooms, and schedules.

Import Data

Restore all system data from a backup file including users, academic management, schedules, and classes.

Clear All Data

Remove all data from the system. This action will sign out all users and cannot be undone.

Create Data Backup

What will be backed up?

  • User accounts and profiles
  • Departments and faculty
  • Subjects and courses
  • Programs and strands
  • Rooms and facilities
  • Schedules and timetables
  • System logs and settings
This backup will be downloaded as a JSON file. Keep it secure as it contains sensitive information.

Clear All Data

This action will permanently delete ALL data from the system

This includes:

  • All user accounts and profiles
  • All departments, faculty, and academic data
  • All subjects, courses, programs, and strands
  • All room assignments and schedules
  • All system logs and configuration data
CRITICAL WARNING: This action cannot be undone. All users will be immediately signed out and redirected to the login page.
Type "CONFIRM" to proceed:

Import Data

What will be imported?

  • Pending user accounts
  • Approved and rejected users
  • Departments and faculty assignments
  • Subjects and courses
  • Programs and strands
  • Rooms and facilities
  • Schedules and timetables
  • Created classes
Warning: Importing data will replace existing data. Make sure you have a backup before proceeding.

Only JSON backup files are supported
